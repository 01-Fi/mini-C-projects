# String Utilities 

## Description
A menu-driven string utilities program written in C that performs
basic string operations using standard string library functions.

## Concepts Used
- Strings
- Arrays
- Standard string functions
- Loops
- switch-case

## Features
- Find string length
- Copy a string
- Reverse a string
- Count vowels in a string

## String Functions Used
- strlen()
- strcpy()

## How to Compile and Run
```bash
gcc string_utils.c -o string_utils
./string_utils

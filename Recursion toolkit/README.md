# Recursion Toolkit 

## Description
A menu-driven recursion toolkit written in C that demonstrates
different recursive algorithms such as factorial, Fibonacci,
sum of digits, and power calculation.

## Concepts Used
- Recursion
- Functions
- Conditional statements
- switch-case
- Loops

## Features
- Factorial calculation
- Fibonacci number calculation
- Sum of digits of a number
- Power calculation using recursion

## How to Compile and Run
```bash
gcc recursion_toolkit.c -o recursion_toolkit
./recursion_toolkit

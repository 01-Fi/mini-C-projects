# Calculator Using Pointers

## Description
A menu-driven calculator program written in C that performs
basic arithmetic operations using functions and call by reference.

## Concepts Used
- Functions
- Pointers (call by reference)
- Loops
- switch-case

## How to Compile and Run
```bash
gcc calculator.c -o calculator
./calculator
